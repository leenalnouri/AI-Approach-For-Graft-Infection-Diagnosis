<b> This repositry contains code from my BSc senior design project, "AI approach for diagnosing graft infections in PET images". </b>

<b> Abstract </b>

<p align="justify">
  Aortic graft infections are post-surgery complications associated with life-threatening
  difficulties which are often diagnosed at a progressive stage. The diagnosis of an aortic graft
  infection is usually made on the basis of clinical findings, supported by radiological and
  microbiological investigations, which are invasive and pose health risks to the patient. This
  project aims to utilize an Artificial Intelligence approach to diagnose graft infections in PET
  images. Due to the infection’s high morbidity and mortality, early identification of AGI is critical
  for immediate treatment. The primary objective of our project is to improve the process of
  diagnosing graft infections by finding a diagnostic ability for AGI using AI with high sensitivity
  and specificity from PET images. Using feature extraction through the LIFEx software as well as
  training, testing and validating the machine learning classification algorithms, the presence of an
  infection was determined. The model was trained with anonymous PET scans to distinguish
  healthy and infected patients using Python programming. Various feature selection algorithms
  were employed to select the most relevant features, followed by classification algorithms which
  will be used for infection prediction. The model with the best diagnostic ability was chosen
  based on the accuracy factors and its feasibility was evaluated for clinical implementation. The
  outcomes of the study offered an AI-based, accurate, faster, and non-invasive method of
  detection. With this design, the priority was improving patient survival rates.
</p>
